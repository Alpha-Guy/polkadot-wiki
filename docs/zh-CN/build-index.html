开发者专页

欢迎访问Polkadot维基网站的构建者部份。

在这里，您可以了解 Polkadot 生态系统中开发工具的最新状态。我们一直致力于学习新的工具和框架，并将它们添加进网站内容中。如您有认为应包含在内的信息，请在Riot上联系我们。

Wiki 的这个部分分为以下:

主页

  • 入门指南 - 有关您的项目应选择平行链，平行线程或智能合约的高级概述，以及现在有哪些资源可以使用。
  • 开发路线图 - 最重要开发工具的状态。
  • 平行链开发套件- PDKs 是用于构建平行链和平行线程的工具套件。
  • 智能合约 - 智能合约是用于制定逻辑使部署到现有的链中。

集成指南

本部分适用于希望学习如何将 Polkadot 集成到其系统中的服务提供商，例如钱包或节点运营商。

  • Polkadot 协议信息 - 关于 Polkadot 协议的一般信息和常见问题如地址格式。
  • 节点管理 - 对 Parity Polkadot 客户端和 最相关的基础设施提供者选项/标志的简短概述。
  • 节点交互-了解如何通过 RPC 与您的节点进行交互。
  • 交易结构和签名 - 了解交易格式以及如何构建离线和签名交易。

工具

  • 工具-维护的工具。
  • Subkey - 用于生成密钥的程序。

资源

  • 例子 - 来自社区的项目 - 查看其他人开发中的项目.

Grants

  • Grants - Web3基金会提供赠款的所有 Polkadot 相关的项目列表。

PSPs

Polkadot 标准建议 (PSP) 是 Polkadot 生态系统的标准。

  • PSP GitHub 库-阅读，创建或讨论标准和提案。
